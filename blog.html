Question

1. What is DOM and what is the purpose of DOM?

Answer: Web documents include a programming interface called the Document Object Model (DOM). Since that software may alter the document's structure, appearance, and content, it serves as a representation of the page.

A website document's elements can be added, changed, or removed using this programming interface. DOM manipulation is the process of adding, deleting, and changing webpage components using JavaScript. In web development, it occurs frequently.

3. What is event bubble?

Answer: Event bubbling is a word you may have come across when researching JavaScript. It refers to the order in which event handlers are invoked when one element is nested inside another and both have assigned a listener for the same action.

4. What is a callback function and why will you use it?

Answer: A callback function is a function that is supplied as an input to another function and then executed inside the outer function to perform some type of procedure or action.

A callback function ensures that a function does not execute before a job is done, but instead runs immediately only after task has been accomplished. It aids in the development of asynchronous JavaScript code and protects us from issues and errors.
